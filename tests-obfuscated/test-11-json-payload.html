REST API Documentation

Authentication

All API requests require authentication using JWT tokens. Include your token in the Authorization header.

POST /api/auth/login

{ "email": "user@example.com", "password": "your_password" }

User Endpoints

GET /api/users/:id

Retrieve user information by ID

{ "id": 123, "name": "John Doe", "email": "john@example.com", "role": "user" }

POST /api/users

Create a new user

{ "name": "Jane Smith", "email": "jane@example.com", "password": "secure_password", "role": "user" }

Response Codes

  • 200 - Success
  • 401 - Unauthorized
  • 404 - Not Found
  • 500 - Server Error
