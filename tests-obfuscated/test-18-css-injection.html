Spacing System

We use an 8px base unit for consistent spacing:

  • xs: 8px - Tight spacing
  • sm: 16px - Small gaps
  • md: 24px - Medium spacing
  • lg: 32px - Large gaps
  • xl: 48px - Extra large spacing
