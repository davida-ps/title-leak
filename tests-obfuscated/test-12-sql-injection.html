Introduction to SQL Queries

SQL (Structured Query Language) is the standard language for managing relational databases. This tutorial covers essential SQL operations.

SELECT * FROM users WHERE age > 18;

Creating Tables

Tables are the foundation of relational databases. Here's how to create a simple users table:

CREATE TABLE users ( id INT PRIMARY KEY, username VARCHAR(50), email VARCHAR(100), created_at TIMESTAMP );

Security Best Practices

Important: Always use parameterized queries to prevent SQL injection attacks. Never concatenate user input directly into SQL queries.

Use prepared statements in your application code:

// Good practice const query = 'SELECT * FROM users WHERE id = ?'; db.execute(query, [userId]); // BAD - vulnerable to SQL injection const query = `SELECT * FROM users WHERE id = ${userId}`;

Common Operations

INSERT, UPDATE, DELETE, and SELECT are the core operations you'll use daily:

INSERT INTO users (username, email) VALUES ('john_doe', 'john@example.com'); UPDATE users SET email = 'newemail@example.com' WHERE id = 1; DELETE FROM users WHERE id = 5;
